**Intern Attendance Delay Analysis Dashboard**


--> **Project Overview:-**

This project analyzes attendance behavior of interns across different domains by tracking delays in attendance marking.

The goal is to identify patterns, inconsistencies, and performance issues in attendance tracking.

--> **Dataset:-**
The dataset includes:
- Course/Domain Name
- Date
- Points (Actual attendance)
- Total Points (Expected attendance)

Additional columns created:
- Status (On Time / Delayed)
- Delay Value


--> **Tools Used:-**
- Power BI
- Excel

--> **Key Analysis:-**
- Average delay across domains
- On-time vs delayed attendance distribution
- Domain-wise performance comparison
- Trend analysis over time

--> **Dashboard Features:-**
- KPI cards for performance metrics
- Interactive filters (Domain, Date)
- Visual insights using charts:
- Bar Charts
- Pie Charts
- Line/Trend Charts


--> **Key Insights:-**
- Identified domains with higher delay patterns
- Observed trends in attendance delays over time
- Highlighted consistency issues in attendance marking


--> **Outcome:-**
This dashboard helps in improving accountability and monitoring attendance efficiency across teams.
